国产良心开发工具：HBuilder

  • 启动速度快
  • 提示全面，自动生成代码，提高编码速度
  • 学好快捷键
  • 边改边看模式直观
  • 查找函数方便
  • 颜色温柔对眼睛友好

SVN

  • 安装后要重启电脑
  • 右键 →SVN Checkout
  • URL of repository 填写仓库路径
  • ...不太熟，以后要注意多学习

GitHub

  • 客户端很难安装好
  • 于是图片要使用外链
  • 网页反应速度特别慢
  • 多次刷新清缓存

码云

N

  • IE兼容到6以上，要根据用户目标人群考虑是否兼容某浏览器
  • CNAME 指定域名
  • master:分支
  • 页面显示代码用 pre 标签
